사용법
현재 유효한 속성은 크게 5가지입니다. tutorial, style, naming, autoPlay, effect입니다.
tutorial
튜토리얼 박스 대상 설정

튜토리얼을 진행 할 대상을 지정하는 리스트입니다. 이때 name속성에 이름을 지정하며, id > class > tagName순으로 대상을 찾습니다. 그리고 message속성에 대상의 설명을 적으면 자동으로 스토리보드가 생성되어 버튼으로 조정되는 튜토리얼이 진행됩니다.

style
튜토리얼 박스 스타일 설정

지정된 대상에 스타일을 조정합니다.

shadowColor는 대상을 가리키는 영역 외 가려지는 부분의 색상을 조정합니다. padding은 대상을 가리키는 영역의 여유공간을 조정합니다.

border, msgBox, btns는 공통으로 rounded속성이 있고, 모서리를 둥글게 합니다.
그 외 border와 msgBox는 color속성을 공통으로 가지고 텍스트 색상을 결정하며, border는 width속성으로 테두리 두께를, msgBox의 bgColor속성으로 메세지 박스의 배경색상을 조정합니다.

naming
튜토리얼 진행 버튼 명칭 설정

버튼별로 이름을 세팅할 수 있도록 했습니다. 고정된 이름보다는 환경에 맞게 변해야 페이지에 조화 될 것이라 생각됩니다. restart는 재시작 버튼의 명칭, prev, next, exit의 버튼 명칭을 변경 할 수 있습니다.
